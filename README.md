# Voxel Canvas

![Preview](https://cdn.poehali.dev/templates/three-fiber-ru/preview-ru.jpg)

Интерактивная 3D-визуализация с воксельным облаком точек на React Three Fiber. Шаблон демонстрирует работу с шейдерами, видеотекстурами и интерактивным управлением камерой через движение мыши. Включает панель управления Leva для настройки параметров рендеринга в реальном времени.
